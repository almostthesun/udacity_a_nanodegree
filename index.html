1. WWW (invented around 1990s)
is a collection of HTML documents. HTML stands for HyperText Markup Language and is the main type of content on the WWW.

HOW IT WORKS

This system of Internet servers supports specially formatted documents. The documents are formatted in the abovementioned language called HTML that supports links to other documents, as well as images, music and video files, pdfs and etc. This means you we can jump from one document to another simply by clicking on hot spots.

Browsers, the Internet, HTTP protocol and Servers are the major pieces that make up the WWW.

BROWSER - a program that runs on a computer to display files found on the Web.
THE INTERNET - the world's largest computer network.
HTTP - the main protocol of the Web
SERVERS - computers that host what make up the Web.

HTML Markup

HTML markup consists of several key components, including tags and their attributes, character-based data types, character references and entity references. HTML tags most commonly come in pairs. Some of them however are called VOID tags as they represent empty elements and so are unpaired. The first tag in such a pair is the opening tag, and the second is the closing tag.

Stupid Computers

To make the first step to successful programing experience one needs to understand that computers are stupid. That is because they interpret instructions literally. This makes them quite unforgiving since even a small typo by a programmer can cause huge problems in a program.

Inline vs Block Elements

HTML elements are either inline or block. Block elements form an "invisible box" around the content inside of them.
